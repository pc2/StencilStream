Detailed Description

template<typename T, uindex_t radius>
class stencil::Stencil< T, radius >

The stencil buffer.

The stencil buffer contains a small chunk of the grid buffer and is used by the transition function to calculate the next-generation cell value.

This implementation provides two ways to index the stencil: With an ID and a UID. Since ID is signed, the column and row axies are within the range of [-radius : radius]. Therefore, (0,0) points to the middle of the stencil. This is the value that is going to be replaced. UID is unsigned and the column and row axies are within the range of [0 : 2*radius + 1). Therefore, (0,0) points to the upper-left corner of the stencil.
