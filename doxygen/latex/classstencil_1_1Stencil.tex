\subsubsection*{template$<$typename T, uindex\+\_\+t radius$>$\newline
class stencil\+::\+Stencil$<$ T, radius $>$}

The stencil buffer.

The stencil buffer contains a small chunk of the grid buffer and is used by the transition function to calculate the next-\/generation cell value.

This implementation provides two ways to index the stencil\+: With an {\ttfamily ID} and a {\ttfamily U\+ID}. Since {\ttfamily ID} is signed, the column and row axies are within the range of \mbox{[}-\/radius \+: radius\mbox{]}. Therefore, (0,0) points to the middle of the stencil. This is the value that is going to be replaced. {\ttfamily U\+ID} is unsigned and the column and row axies are within the range of \mbox{[}0 \+: 2$\ast$radius + 1). Therefore, (0,0) points to the upper-\/left corner of the stencil.
