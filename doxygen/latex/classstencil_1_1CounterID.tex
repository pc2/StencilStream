\doxysubsection{Detailed Description}
\subsubsection*{template$<$typename T$>$\newline
class stencil\+::\+Counter\+I\+D$<$ T $>$}

A subclass of the {\ttfamily \mbox{\hyperlink{classstencil_1_1GenericID}{Generic\+ID}}$<$T$>$} with two-\/dimensional index counting. 

It additionally requires bounds for the column and row counters and provides increment and decrement operators. The row counter is always the fastest\+: For example, the increment operators first increase the row counter and when it reaches the row bound, it is reset to zero and the column counter is increased. When the column counter reaches it\textquotesingle{}s bound, it is also reset to zero. One can therefore use this counter to safely iterate over a two-\/dimensional grid even with multiple passes.
