Detailed Description

The requirements for a pass-local TDV system state.

Each local state is instantiated just prior to starting a pass, and the final time-dependent values are extracted from it.

The required methods are:

  • TDV get_time_dependent_value(uindex_t i) const: Return the time-dependent value for the pass-local iteration i. This is the current iteration minus the iteration index offset of the current pass. The type TDV has to be the same as TransFunc::TimeDependentValue.
