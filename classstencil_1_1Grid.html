Detailed Description

template<typename T, uindex_t tile_width, uindex_t tile_height, uindex_t halo_radius, uindex_t burst_length>
class stencil::Grid< T, tile_width, tile_height, halo_radius, burst_length >

A rectangular container of cells with a dynamic, arbitrary size.

It logically contains the grid the transition function is applied to. As described in Grid Tiling, it partitions the grid into tiles of static size, which are the units the ExecutionKernel works on.

Apart from providing copy operations to and from monolithic grid buffers, it also handles the input and output kernel submission for a given tile.

Template Parameters
TCell value type.
tile_widthThe number of columns of a tile.
tile_heightThe number of rows of a tile.
halo_radiusThe radius (aka width and height) of the tile halo.
burst_lengthThe number of elements that can be read or written in a burst.

inline

Return the range of (central) tiles of the grid.

This is not the range of a single tile nor is the range of the grid. It is the range of valid arguments for Grid::get_tile. For example, if the grid is 60 by 60 cells in size and a tile is 32 by 32 cells in size, the tile range would be 2 by 2 tiles.

Returns
The range of tiles of the grid.
